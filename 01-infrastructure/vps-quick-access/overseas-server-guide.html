二、路子一：AWS $200 Free Credits 完整路径

2.1 方案总览

AWS 于 2025 年 7 月 15 日 更新了 Free Tier 政策。新用户注册即可获取 $100 信用金，完成 5 个入门活动再获 $100，合计 $200。

信用金可用于所有 AWS 服务（包括 Lightsail、EC2），有效期 6 个月（或消费完为止）。到期后若未升级为付费账户，账号将自动注销。

2.2 注册与获取 $200 的完整步骤

准备工作

  • 一个未注册过 AWS 的邮箱（Gmail/Outlook 均可）
  • 一张支持国际支付的 Visa/Mastercard 信用卡或借记卡
  • 一个能接收国际短信的手机号
1
访问 AWS Free Tier 注册页
打开 https://aws.amazon.com/free/，点击 Create a Free Account。
2
验证邮箱
输入邮箱和账户名。AWS 会发送验证码到邮箱，输入验证码完成验证。
3
设置 root 用户密码
设置一个强密码（至少 8 位，包含大小写字母、数字、特殊符号）。
4
填写联系信息
姓名、地址、城市、邮编——填写真实信息或用拼音。选择 Personal account。
5
绑定信用卡/借记卡
输入卡号、有效期、CVV。AWS 会进行 $1 预授权验证（会退还）。
仅支持 Visa/Mastercard
6
验证手机号
选择国家代码 (+86)，输入手机号，接听 AWS 自动来电或输入短信验证码。
7
选择支持计划
选择 Basic support (Free)，不要选付费支持计划。
8
注册完成，$100 自动到账
登录 AWS Console，在右上角账户菜单中可看到 Credits 余额。

2.3 完成 5 个活动，再拿 $100

登录 AWS Console 后，在 Dashboard 中找到 "Explore AWS" 小部件，依次完成以下 5 个活动。每个活动奖励 $20 credits。
#活动名称操作描述难度
1 Launch an EC2 instance 在 EC2 控制台启动一个 t2.micro 实例，完成后 立即终止 避免扣费 ⭐ 简单
2 Use Amazon Bedrock playground 进入 Bedrock 演练场，选择一个模型（如 Claude），输入一段 prompt 获取回复 ⭐ 简单
3 Set up AWS Budgets 创建一个零成本预算模板，监控账户消费 ⭐ 简单
4 Create a Lambda web app 创建一个 Lambda 函数并配置 Function URL，返回 "Hello World" ⭐⭐ 中等
5 Create an RDS database 启动一个 RDS 或 Aurora 数据库实例，浏览基本配置后删除 ⭐⭐ 中等
完成活动会产生少量服务费用，直接从 Credits 中扣除。做完任务记得清理资源（终止实例、删除数据库）。

2.4 用 Credits 开 Lightsail / EC2 部署服务

选项 A：Lightsail（推荐新手）

Lightsail 是 AWS 的「轻量云服务器」，套餐包含计算、存储、流量，没有隐藏费用。

套餐vCPU内存SSD流量/月价格Credits 可跑
$5/mo21 GB40 GB2 TB$5/月40 个月*
$12/mo22 GB60 GB3 TB$12/月16 个月*
$24/mo24 GB80 GB4 TB$24/月8 个月*

* 实际受 6 个月有效期限制。推荐 $12/mo 方案，2GB 内存足够跑 gridbot + 若干轻量服务。

1
进入 Lightsail 控制台
https://lightsail.aws.amazon.com/
2
创建实例
点击 Create instance → 选择区域（选新加坡 ap-southeast-1 或日本 ap-northeast-1 延迟更低）→ 选择 Linux/Unix 平台 → 选择 Ubuntu 22.04 LTS → 选择 $12 USD/mo 套餐。
3
命名并创建
输入实例名称（如 gridbot-server）→ 点击 Create instance。
4
配置防火墙与静态 IP
进入实例详情页 → Networking → 添加需要的端口（如 22, 80, 443, 8080）→ 点击 Create static IP 分配固定公网 IP（免费，实例运行期间不收费）。
5
SSH 登录 & 部署
ssh -i ~/.ssh/lightsail-key.pem ubuntu@your-static-ip

# 更新系统
sudo apt update && sudo apt upgrade -y

# 安装 Docker（方便部署各种服务）
curl -fsSL https://get.docker.com | sudo bash
sudo usermod -aG docker ubuntu

# 部署 gridbot 或其他服务
# docker run -d ...

选项 B：EC2（更灵活）

EC2 可定制性更强，但需要同时管理 EBS 存储、安全组、弹性 IP 等。适合对 AWS 有一定了解的用户。

实例类型vCPU内存约月费说明
t3.micro21 GB~$7.5/mo (含 30GB EBS)Free Tier 12个月免费（但需 2025.7 前注册）
t3.small22 GB~$15/mo (含 30GB EBS)适合轻量服务
t3.medium24 GB~$30/mo (含 30GB EBS)适合中等负载

用 $200 credits 开 t3.small 约可跑 13 个月（但受 6 个月有效期限制）。

建议：新手选 Lightsail（简单、无隐藏费用、自带流量包）；有经验选 EC2（可定制安全组、VPC、Auto Scaling 等）。

2.5 AWS 方案注意事项

重要提醒：
  • 6 个月期满前务必迁移——账户到期后会自动注销，所有数据将丢失
  • 如果想继续使用，需在到期前升级为付费账户，之后按实际用量付费
  • Lightsail $12/mo 方案 6 个月消费 $72，200 credits 绰绰有余
  • 完成 5 个活动时会产生服务费，记得关闭非必要资源
  • 免费静态 IP 仅在运行中的实例上免费，实例停止后 IP 会被释放
  • 默认配额限制：新账户 EC2 实例数不超过 5 个
  • 注册必须使用 信用卡/借记卡，不接受预付卡或虚拟卡

# SSH 登录

apt update && apt upgrade -y

curl -fsSL https://get.docker.com | bash

# 安装 Docker Compose
apt install docker-compose-plugin -y

# 创建项目目录
mkdir -p /opt/gridbot
cd /opt/gridbot

# 创建 docker-compose.yml，部署你的 gridbot 等服务

# docker compose up -d

ufw allow 22 && ufw allow 80 && ufw allow 443 && ufw enable

3.7 HostDzire 注意事项

  • 促销价波动大——价格取决于 LET 帖子的优惠码，一旦过期可能恢复原价
  • 印度商家——客服支持有时差，英文沟通为主
  • 不支持退款——购买前请确认需求
  • PayPal 支付比信用卡更安全，建议使用 PayPal
  • IP 可能被墙——如果服务面向国内用户，建议购买后先测试 IP 连通性，若被墙可开工单申请换 IP
  • 备份重要——低价 VPS 不保证数据安全，建议定期备份到外部存储
  • CPU 非独享——10Gbps 共享带宽、共享 CPU，邻居滥用可能影响性能
