## 包含的 Skills

| Skill | 功能 |
|-------|------|
| [tech-doc-organizer](./skills/tech-doc-organizer/) | 技术驱动型开发文档组织规范，辅助生成技术设计文档 |

---

## 使用方式

### tech-doc-organizer

#### 触发条件

用户说"整理成技术文档"、"沉淀成文档"、"生成设计文档"、"按规范整理"时触发。

#### 前置条件

使用本 Skill 前，需要明确：
1. **任务类型**：具体开发任务、Bug 修复、技术调研、功能实现
2. **输出目录**：用户指定的文档存放目录

#### 推荐用法

技术实现类任务使用此 Skill 辅助生成完整的技术文档集。
