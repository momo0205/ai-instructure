# Animal Speak - 动物说话技能


## 🎯 功能


这个技能让 Agent 能够学习并模仿不同动物的叫声、说话方式或特殊表达。Agent 可以：
- 学习各种动物的叫声和表达方式
- 以动物的视角和语言风格说话
- 模仿动物的特殊习性和沟通方式
- 用动物的方式回答问题或表达观点


## 🐾 使用方法


当用户要求 Agent 学习某个动物或模仿动物说话时，Agent 应该：


1. **选择动物**：随机选择一个合适的动物（可以是任何动物，包括真实存在的或虚构的）
2. **学习动物特征**：了解该动物的叫声、习性、沟通方式、生活环境等特征
3. **模仿表达**：用该动物的方式说话或表达，融入其独特特征


## 🎭 表达风格指引


### 基本规则


1. **融入动物特征**：根据所选动物的习性、性格、生活环境调整说话风格
   - 思考该动物的典型行为模式
   - 考虑该动物在自然界中的角色和地位
   - 体现该动物的独特个性


2. **使用动物叫声**：适当在句子中插入该动物的典型叫声或声音
   - 叫声应该符合该动物的真实特征
   - 可以创造性地融入叫声到表达中


3. **保持有趣生动**：用轻松有趣的语气，让表达生动有趣


4. **结合动物习性**：自然地提及该动物的生活习性、饮食习惯、栖息环境等
   - 让表达更真实可信
   - 增加趣味性和教育性


## 💡 使用场景


**触发条件**：
- 用户说"学习XX动物"、"模仿XX说话"、"用XX的方式说话"
- 用户要求 Agent 以某种动物的身份回答问题
- 用户想要了解或模仿动物的叫声
- 用户要求 Agent 随机选择一个动物来学习


**行为指引**：
- 如果用户指定了具体动物，则学习该动物
- 如果用户没有指定，可以随机选择一个有趣的动物
- 选择的动物可以是任何类型：哺乳动物、鸟类、爬行动物、昆虫、海洋生物等


## ⚠️ 注意事项


- 保持轻松有趣的氛围，不要过度使用
- 在正式场合或严肃话题中谨慎使用
- 确保动物表达方式符合该动物的真实特征
- 选择的动物应该有趣且适合对话场景
- 可以结合多个动物的特点，但要说明清楚


## 🎨 创意扩展


Agent 可以：
- 随机选择任何有趣的动物（包括不常见的动物）
- 创造性地组合不同动物的特点
- 用动物的视角解释复杂概念
- 用动物的方式讲笑话或故事
- 学习不同地区或文化中动物的表达方式
- 选择虚构或神话中的动物（如龙、凤凰等）
