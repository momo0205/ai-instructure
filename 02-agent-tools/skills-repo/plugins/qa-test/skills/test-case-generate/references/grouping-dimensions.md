# 分组维度指南

## 分组维度总览

| 分组类型 | 适用场景 | 典型覆盖内容 | 优先级 |
|---------|---------|-------------|--------|
| 显示条件 | 有条件显示/隐藏的组件 | 显示/隐藏条件、位置关系 | 高 |
| 状态表现 | 有多状态的组件 | 每状态的文案、视觉、交互 | 高 |
| 状态流转 | 状态机组件 | 正常流转、异常回退、刷新恢复 | 高 |
| 交互行为 | 可交互组件 | 点击、hover、禁用状态 | 高 |
| 内容展示 | 内容/预览区域 | 渲染、刷新、错误兜底 | 中 |
| 全局通知 | 有 Notification/Toast | 成功/失败提示、点击行为 | 中 |
| 状态同步 | 多组件联动 | 跨组件状态一致性 | 中 |
| 多语言 | 有 i18n 支持 | 中英文切换验证 | 中 |

## 1. 显示条件

**适用场景**：组件根据某些条件显示或隐藏

**典型测试点**：

```markdown
### ▼ 1. 显示条件

- [ ] 1.1 [条件A] 时，显示 [组件]
- [ ] 1.2 [条件B] 时，不显示 [组件]
- [ ] 1.3 [组件] 位于 [参照物] 的 [位置关系]
- [ ] 1.4 [条件变化] 时，显示状态正确切换
```

## 2. 状态表现

**适用场景**：组件有多个状态（如 idle/loading/success/error）

**每个状态需要覆盖 4 个维度**：

| 维度 | 说明 | 示例 |
|------|------|------|
| 文案 | 按钮/标签的文字 | `按钮文案显示「XX」` |
| 视觉 | 样式、动效 | `显示加载动效` / `按钮置灰` |
| 点击 | 点击后的响应 | `点击触发 XX` / `禁用无反应` |
| Hover | 鼠标悬停效果 | `Tooltip「XX」` + `鼠标指针样式` |

**模板**：

```markdown
### ▼ 2. [状态名] 状态

- [ ] 2.1 [元素] 文案显示「XX」
- [ ] 2.2 显示 [视觉效果]
- [ ] 2.3 点击 [行为]
- [ ] 2.4 hover 显示 Tooltip「XX」
- [ ] 2.5 hover 鼠标 [显示/不显示] 小手指针
```

## 3. 状态流转

**适用场景**：组件状态会随操作/事件变化

**典型测试点**：

```markdown
### ▼ 3. 状态流转

- [ ] 3.1 [操作] 后，状态从 [A] 变为 [B]（正常流程）
- [ ] 3.2 [异常] 后，状态从 [B] 回退到 [A]（异常回退）
- [ ] 3.3 页面刷新后，状态正确恢复（刷新恢复）
- [ ] 3.4 切换 [上下文] 后，状态正确切换
```

## 4. 交互行为

**适用场景**：需要特殊处理的交互细节

**典型测试点**：

```markdown
### ▼ 4. 交互行为

- [ ] 4.1 点击按钮，不触发父容器的 [事件]（事件冒泡）
- [ ] 4.2 [禁用状态] 点击，无任何反应
- [ ] 4.3 快速连续点击，只触发一次（防抖）
```

## 5. 内容展示

**适用场景**：内容区域、预览面板

**典型测试点**：

```markdown
### ▼ 5. 内容展示

- [ ] 5.1 内容正确渲染（格式、样式）
- [ ] 5.2 内容刷新后自动更新
- [ ] 5.3 加载失败时，显示错误兜底页
```

## 6. 全局通知

**适用场景**：有 Notification/Toast 提示

**典型测试点**：

```markdown
### ▼ 6. 全局通知

- [ ] 6.1 [成功] 时显示通知，标题「XX」
- [ ] 6.2 通知包含按钮「XX」
- [ ] 6.3 点击通知按钮，执行 [操作]
- [ ] 6.4 [失败] 时显示错误通知
```

## 7. 状态同步

**适用场景**：多个组件状态需要联动

**典型测试点**：

```markdown
### ▼ 7. 状态同步

- [ ] 7.1 [组件A] 触发操作时，[组件B] 同步进入 [状态]
- [ ] 7.2 操作完成后，两组件同步恢复正常
```

## 8. 多语言

**适用场景**：支持 i18n 的组件

**典型测试点**：

```markdown
### ▼ N. 多语言

- [ ] N.1 切换英文后，[状态A] 显示「English」
- [ ] N.2 切换英文后，Tooltip 显示「English tip」
- [ ] N.3 切换英文后，通知标题显示「English title」
```

## 分组选择决策树

```
组件有条件显示？ ────→ 添加「显示条件」分组
         │
组件有多个状态？ ────→ 为每个状态添加分组
         │
状态会流转变化？ ────→ 添加「状态流转」分组
         │
有特殊交互细节？ ────→ 添加「交互行为」分组
         │
有内容/预览区域？ ───→ 添加「内容展示」分组
         │
有全局通知提示？ ────→ 添加「全局通知」分组
         │
多组件需要联动？ ────→ 添加「状态同步」分组
         │
支持多语言？ ────────→ 添加「多语言」分组
```
