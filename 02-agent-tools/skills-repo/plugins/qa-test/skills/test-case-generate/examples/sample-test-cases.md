# 示例：测试用例输出格式

> **说明**：本示例展示完整的测试用例输出格式和风格，可作为参考模板。
> 
> 示例来源于一个实际的交互设计文档（Agent Review 功能），展示了：
> - 场景梳理的 `「场景名」→` 格式
> - 功能点测试的分组和编号
> - 状态测试的四个维度（文案、视觉、点击、hover）
> - 多语言单独分组
> 
> **实际使用时**：根据具体的设计文档内容来定制组件名、状态名、文案等。

## 📋 hx对话面板工作区 - 测试用例

### 场景梳理

- [ ] 0.1 「首次生成」工作区有文件→点击「智能审查」→按钮变为「审查中…」（扫光动效）→生成完成→按钮变为「查看审查结果」→右下角弹出成功通知
- [ ] 0.2 「查看报告」已有报告→点击「查看审查结果」→编辑器区域打开报告 Tab→正确显示 Markdown 内容
- [ ] 0.3 「通知跳转」生成完成→右下角弹出通知→点击通知「查看审查结果」按钮→打开报告 Tab
- [ ] 0.4 「生成失败重试」点击「智能审查」→生成失败→按钮恢复「智能审查」→右下角弹出失败通知→点击「重新审查」→重新触发生成
- [ ] 0.5 「生成中禁用」点击「智能审查」→进入生成中→再次点击按钮→无反应（禁用）
- [ ] 0.6 「生成中不折叠」点击「智能审查」→进入生成中→点击按钮→工作区不折叠
- [ ] 0.7 「刷新恢复状态-有报告」已有报告→刷新页面→按钮显示「查看审查结果」
- [ ] 0.8 「刷新恢复状态-无报告」无报告→刷新页面→按钮显示「智能审查」
- [ ] 0.9 「切换 Thread」Thread A 有报告，Thread B 无报告→从 A 切换到 B→按钮从「查看审查结果」变为「智能审查」

### ▼ 1. 按钮显示条件

- [ ] 1.1 工作区有文件时，工作区条显示「智能审查」按钮
- [ ] 1.2 工作区无文件时，不显示「智能审查」按钮
- [ ] 1.3 工作区有文件但所有文件已 Accept/Reject 时，仍显示「智能审查」按钮
- [ ] 1.4 「智能审查」按钮位于「全部拒绝」按钮左侧

### ▼ 2. idle 状态（未生成过报告）

- [ ] 2.1 未生成过报告时，按钮文案显示「智能审查」
- [ ] 2.2 hover 按钮时，显示 Tooltip「点击后，将根据已修改的文件内容进行智能审查」
- [ ] 2.3 hover 按钮时，鼠标显示小手指针
- [ ] 2.4 点击按钮，触发 Review 生成，按钮进入 loading 状态

### ▼ 3. loading 状态（生成中）

- [ ] 3.1 生成中按钮文案显示「审查中…」
- [ ] 3.2 生成中按钮显示扫光动效
- [ ] 3.3 生成中点击按钮无反应（按钮禁用）
- [ ] 3.4 生成中 hover 按钮，显示 Tooltip「Review 生成中请等待」
- [ ] 3.5 生成中 hover 按钮，鼠标不显示小手指针
- [ ] 3.6 生成中点击按钮，不触发工作区折叠/展开

### ▼ 4. ready 状态（已有报告）

- [ ] 4.1 生成完成后，按钮文案变更为「查看审查结果」
- [ ] 4.2 hover 按钮时，鼠标显示小手指针
- [ ] 4.3 点击「查看审查结果」，在编辑器区域打开报告 Tab

### ▼ 5. 状态流转

- [ ] 5.1 首次点击生成：idle「智能审查」→ loading「审查中…」→ ready「查看审查结果」
- [ ] 5.2 生成失败后，按钮恢复为 idle 状态「智能审查」
- [ ] 5.3 页面刷新后，若有历史报告，按钮显示为 ready 状态「查看审查结果」
- [ ] 5.4 页面刷新后，若无历史报告，按钮显示为 idle 状态「智能审查」
- [ ] 5.5 切换 Thread 后，按钮状态正确切换为对应 Thread 的状态

### ▼ 6. 全局通知（Notification）

- [ ] 6.1 生成成功后，右下角显示通知，标题「智能审查完成」
- [ ] 6.2 生成成功通知显示说明「智能审查已完成，点击打开查看审查结果」
- [ ] 6.3 生成成功通知显示按钮「查看审查结果」
- [ ] 6.4 点击通知「查看审查结果」按钮，打开报告
- [ ] 6.5 生成失败后，右下角显示通知，标题「智能审查失败」
- [ ] 6.6 生成失败通知显示按钮「重新审查」
- [ ] 6.7 点击通知「重新审查」按钮，重新触发审查

### ▼ 7. 多语言

- [ ] 7.1 切换英文后，idle 状态按钮文案显示「Agent Review」
- [ ] 7.2 切换英文后，loading 状态按钮文案显示「Reviewing…」
- [ ] 7.3 切换英文后，ready 状态按钮文案显示「View Analysis」
- [ ] 7.4 切换英文后，idle 状态 hover Tooltip 显示「Runs an agent-based review on your latest file changes.」
- [ ] 7.5 切换英文后，生成成功通知标题显示「Agent Review Complete」
- [ ] 7.6 切换英文后，生成成功通知说明显示「The agent-based review has finished. Click to view the results.」
- [ ] 7.7 切换英文后，生成成功通知按钮显示「View Results」
- [ ] 7.8 切换英文后，生成失败通知标题显示「Agent Review Failed」
- [ ] 7.9 切换英文后，生成失败通知按钮显示「Retry」

## 📋 Review Report 报告展示区 - 测试用例

### 场景梳理

- [ ] 0.1 「重新生成-有更新」打开报告→点击「再次审查」→两个按钮同步进入 loading→生成完成→报告内容自动刷新→弹出成功通知
- [ ] 0.2 「重新生成-无变化」打开报告→代码无修改→点击「再次审查」→生成完成→报告内容不变→弹出「生成内容没有变化」通知
- [ ] 0.3 「重新生成失败」打开报告→点击「再次审查」→生成失败→显示错误兜底页→「再次审查」按钮仍可点击→点击重试
- [ ] 0.4 「状态同步」打开报告→点击「再次审查」→工作区条按钮同步显示扫光动效→生成完成→两个按钮同步恢复正常
- [ ] 0.5 「生成中仍可查看」已有报告→从 ReviewAgainButton 触发重新生成→生成中点击工作区「查看审查结果」→仍可打开报告查看旧内容

### ▼ 1. 面板打开与展示

- [ ] 1.1 点击「查看审查结果」后，在编辑器区域打开新 Tab
- [ ] 1.2 Tab 名称显示为「Agent review」
- [ ] 1.3 面板正确渲染 Markdown 格式的报告内容
- [ ] 1.4 报告包含 Summary 概述
- [ ] 1.5 报告包含 Issue 列表，按 P0/P1/P2 分级展示
- [ ] 1.6 Issue 条目包含编号、标题、状态、影响文件、描述、建议
- [ ] 1.7 报告中的代码块语法高亮正常
- [ ] 1.8 多次打开同一报告，内容一致

### ▼ 2. ReviewAgainButton 状态

- [ ] 2.1 ReviewAgainButton 位于面板左上角悬浮显示
- [ ] 2.2 normal 状态按钮文案显示「再次审查」
- [ ] 2.3 normal 状态 hover 显示 Tooltip「重新生成 Review」
- [ ] 2.4 点击「再次审查」，触发重新生成，按钮进入 loading 状态
- [ ] 2.5 loading 状态按钮文案显示「审查中...」
- [ ] 2.6 loading 状态按钮显示 Loading 动画（SVG 旋转）
- [ ] 2.7 loading 状态按钮禁用，点击无反应
- [ ] 2.8 loading 状态 hover 显示 Tooltip「Review 生成中...」
- [ ] 2.9 生成完成后，按钮恢复为 normal 状态「再次审查」
- [ ] 2.10 生成失败后，按钮仍恢复为 normal 状态「再次审查」（仍可用）

### ▼ 3. 内容刷新

- [ ] 3.1 点击「再次审查」后，代码有变更时，报告内容自动刷新
- [ ] 3.2 报告刷新无需手动关闭再打开面板
- [ ] 3.3 代码无变化时重新生成，显示通知「生成内容没有变化」
- [ ] 3.4 代码无变化时重新生成，报告内容保持不变

### ▼ 4. 错误兜底

- [ ] 4.1 生成失败（超时/异常）时，面板显示错误兜底页
- [ ] 4.2 错误兜底页中，ReviewAgainButton 仍可用
- [ ] 4.3 在错误兜底页点击「再次审查」，可重新触发生成

### ▼ 5. 状态同步

- [ ] 5.1 点击 ReviewAgainButton 时，工作区条按钮同步进入 loading 状态（扫光动效）
- [ ] 5.2 重新生成期间，工作区条按钮仍可点击打开面板（ready + loading 叠加）
- [ ] 5.3 生成完成后，两个按钮同步恢复正常状态
- [ ] 5.4 生成失败后，两个按钮同步恢复正常状态

### ▼ 6. 多语言

- [ ] 6.1 切换英文后，ReviewAgainButton normal 状态显示「Review Again」
- [ ] 6.2 切换英文后，ReviewAgainButton loading 状态显示「Reviewing...」
- [ ] 6.3 切换英文后，normal 状态 hover Tooltip 显示「Regenerate Review」
- [ ] 6.4 切换英文后，loading 状态 hover Tooltip 显示「Review in progress...」
